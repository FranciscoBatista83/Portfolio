# Meu Portfólio Pessoal(em constante evolução)

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido com HTML, CSS. O site foi criado para exibir informações sobre mim, meus projetos, formação e formas de contato.

## Funcionalidades

- Seção "Sobre Mim" com descrição pessoal.
- Exibição de projetos com detalhes e links.
- Seção de formação acadêmica.
- Formulário de contato.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Responsive Design (Mobile First)
